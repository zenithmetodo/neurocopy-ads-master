# /ads-guion · El cuerpo del anuncio con estructura (por nivel)

Lanza `guion-body-master`: coge un ángulo + nivel + formato y escribe el **cuerpo palabra por palabra** con una **estructura de guion** del nivel correspondiente (incluido **HARD SELL** en nivel alto), el **PUENTE** (gancho→cuerpo gradual) y timestamps `DI →`. Mejora la estructura base, no la copia plana.

## Cómo usarlo
```
/ads-guion
/ads-guion escríbeme el cuerpo en HARD SELL para el anuncio de nivel alto del ángulo 2
/ads-guion dame 3 cuerpos distintos (PCMISUC · CON-vs-SIN · demostrativa) para este ángulo en nivel medio
```

## Qué entrega
- Estructura usada (B/M/A-x) + por qué encaja + el **PUENTE** + el **cuerpo palabra por palabra** (timestamps) + mejoras aplicadas + por qué funciona (interno).
